# 使用结构赋值，实现两个变量的值的交换（高薪常问）

**题目**: 使用结构赋值，实现两个变量的值的交换（高薪常问）
### 标准答案

使用ES6的解构赋值可以非常简洁地实现两个变量值的交换，语法为 `[a, b] = [b, a]`。这种方式无需借助临时变量，代码更简洁易读。

### 深入理解

解构赋值是ES6引入的语法糖，它允许我们从数组或对象中提取值并赋给变量。对于变量交换，其核心原理是同时对多个变量进行赋值操作。

**基本用法：**
```javascript
let a = 1;
let b = 2;

// 使用解构赋值交换值
[a, b] = [b, a];

console.log(a); // 输出: 2
console.log(b); // 输出: 1
```

**更复杂的交换场景：**
```javascript
// 交换多个变量
let x = 1, y = 2, z = 3;
[x, y, z] = [z, x, y]; // x=3, y=1, z=2

// 交换数组中的元素
let arr = [1, 2, 3, 4];
[arr[0], arr[3]] = [arr[3], arr[0]]; // 交换首尾元素
console.log(arr); // [4, 2, 3, 1]

// 交换对象属性
let obj1 = { value: 10 };
let obj2 = { value: 20 };
[obj1.value, obj2.value] = [obj2.value, obj1.value];
console.log(obj1.value); // 20
console.log(obj2.value); // 10
```

**与其他交换方式的对比：**
```javascript
// 传统方式 - 使用临时变量
let temp = a;
a = b;
b = temp;

// 数学运算方式（仅适用于数字）
a = a + b;
b = a - b;
a = a - b;

// 位运算方式（仅适用于整数）
a = a ^ b;
b = a ^ b;
a = a ^ b;

// 解构赋值方式（推荐）
[a, b] = [b, a]; // 最简洁，可读性最好
```

**注意事项：**
1. 解构赋值交换时，右侧的值会被先计算，然后同时赋给左侧的变量
2. 这种方式不仅适用于基本类型，也适用于对象和数组
3. 在某些情况下，如解构赋值中有计算或函数调用，需要注意执行顺序
