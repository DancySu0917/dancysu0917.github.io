# H5 的新特性有哪些？C3 的新特性有哪些？（必会）

## HTML5 的新特性

### 1. 语义化标签
- `<header>`: 页面或章节的头部
- `<nav>`: 导航链接
- `<section>`: 文档中的区段
- `<article>`: 独立的文章内容
- `<aside>`: 侧边栏或附属信息
- `<footer>`: 页面或章节的底部
- `<main>`: 页面的主要内容

### 2. 多媒体支持
- `<audio>`: 音频播放
- `<video>`: 视频播放
- `<source>`: 媒体资源的源文件

### 3. 表单增强
- 新的输入类型：`email`, `url`, `date`, `time`, `number`, `range`, `search`, `tel` 等
- 新的表单属性：`placeholder`, `required`, `autofocus`, `autocomplete`
- 表单验证 API

### 4. 图形绘制
- `<canvas>`: 2D 图形绘制
- `<svg>`: 可缩放矢量图形

### 5. 存储功能
- localStorage: 本地持久存储
- sessionStorage: 会话存储
- IndexedDB: 浏览器数据库
- WebSQL: 结构化数据库（已废弃）

### 6. 网络通信
- WebSocket: 全双工通信
- Server-Sent Events: 服务器推送事件
- XMLHttpRequest Level 2

### 7. 地理定位
- Geolocation API: 获取用户地理位置

### 8. 硬件访问
- File API: 文件操作
- Drag & Drop API: 拖拽功能
- Web Workers: 后台线程处理

### 9. 离线应用
- Application Cache: 离线缓存
- Service Workers: 更强大的离线功能

### 10. 其他 API
- History API: 浏览历史管理
- Web Messaging: 跨文档通信
- Web Storage: 存储接口


## CSS3 的新特性

### 1. 选择器增强
- 属性选择器：`[attr^="value"]`, `[attr$="value"]`, `[attr*="value"]`
- 伪类选择器：`:nth-child()`, `:not()`, `:first-of-type`, `:last-of-type`
- 伪元素选择器：`::before`, `::after`, `::first-letter`, `::first-line`

### 2. 边框效果
- `border-radius`: 圆角边框
- `box-shadow`: 盒阴影
- `border-image`: 边框图片

### 3. 背景和边框
- 多背景图片
- `background-size`: 背景尺寸
- `background-origin`: 背景原点
- `background-clip`: 背景裁剪

### 4. 渐变效果
- 线性渐变：`linear-gradient()`
- 径向渐变：`radial-gradient()`
- 重复渐变：`repeating-linear-gradient()`

### 5. 文本效果
- `text-shadow`: 文本阴影
- `word-wrap`: 文本换行
- `text-overflow`: 文本溢出
- `@font-face`: 自定义字体

### 6. 2D/3D 变换
- `transform`: 旋转、缩放、移动、倾斜
- `transform-origin`: 变换原点
- 3D 变换函数

### 7. 过渡和动画
- `transition`: 过渡效果
- `animation`: 动画效果
- `keyframes`: 关键帧定义

### 8. 弹性布局
- Flexbox: 弹性盒模型
- `display: flex`
- 相关属性：`flex-direction`, `justify-content`, `align-items` 等

### 9. 网格布局
- CSS Grid Layout
- `display: grid`
- 网格容器和网格项目属性

### 10. 多列布局
- `column-count`: 列数
- `column-gap`: 列间距
- `column-rule`: 列分割线

### 11. 用户界面
- `resize`: 元素可调整大小
- `box-sizing`: 盒模型计算方式
- `outline-offset`: 轮廓偏移

### 12. 颜色
- RGBA/HSLA: 带透明度的颜色
- `opacity`: 元素透明度
