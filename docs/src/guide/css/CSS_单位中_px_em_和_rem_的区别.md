# CSS 单位中 px、em 和 rem 的区别？（必会）

**题目**: CSS 单位中 px、em 和 rem 的区别？（必会）

**答案**:

CSS中的px、em和rem是三种常用的长度单位，它们有着不同的计算方式和应用场景。

## 1. px (像素)

### 定义
- px是绝对单位，代表屏幕上的一个物理像素点
- 1px通常等于屏幕上的一个像素点

### 特点
- 固定大小，不会因为父元素或根元素的字体大小而改变
- 浏览器默认缩放比例下，16px ≈ 1em (基于浏览器默认字体大小16px)

### 优点
- 精确控制元素尺寸
- 计算简单，易于理解

### 缺点
- 不够灵活，无法响应用户缩放设置
- 不利于可访问性（用户调整浏览器字体大小时，px单位的元素不会相应变化）

### 使用场景
- 固定尺寸的元素，如边框、阴影等
- 需要精确控制的场景

```css
.container {
    width: 300px;      /* 固定300像素宽度 */
    font-size: 16px;   /* 固定16像素字体大小 */
    border: 1px solid #ccc;  /* 固定1像素边框 */
}
```

## 2. em

### 定义
- em是相对单位，相对于当前元素的字体大小
- 如果当前元素未设置字体大小，则相对于父元素的字体大小

### 特点
- 1em = 当前元素的font-size值
- 具有继承性，子元素的em值会基于父元素的字体大小计算

### 优点
- 响应用户字体大小设置
- 保持元素间比例关系

### 缺点
- 容易产生嵌套问题（嵌套元素的em会基于父元素计算，导致计算复杂）

### 使用场景
- 需要与字体大小保持比例的元素（如内边距、边距等）

```css
.container {
    font-size: 16px;   /* 基准字体大小 */
}

.child {
    width: 20em;       /* 20 × 16px = 320px */
    padding: 1em;      /* 1 × 16px = 16px */
}

.nested {
    font-size: 1.2em;  /* 1.2 × 16px = 19.2px */
    margin: 1.5em;     /* 1.5 × 19.2px = 28.8px */
}
```

## 3. rem

### 定义
- rem是相对单位，相对于根元素（html元素）的字体大小
- "rem"代表"root em"

### 特点
- 1rem = 根元素的font-size值
- 不受父元素字体大小影响，始终基于根元素

### 优点
- 保持响应性，同时避免了em的嵌套问题
- 便于整体缩放控制

### 缺点
- 需要设置根元素字体大小才能有效利用

### 使用场景
- 响应式设计
- 需要整体缩放控制的场景
- 现代Web开发的推荐选择

```css
html {
    font-size: 16px;   /* 根元素字体大小 */
}

.container {
    width: 20rem;      /* 20 × 16px = 320px */
    padding: 1rem;     /* 1 × 16px = 16px */
    font-size: 1.2rem; /* 1.2 × 16px = 19.2px */
}

.nested {
    margin: 1.5rem;    /* 1.5 × 16px = 24px，不受父元素影响 */
}
```

## 三者对比

| 特性 | px | em | rem |
|------|----|----|-----|
| 类型 | 绝对单位 | 相对单位 | 相对单位 |
| 基准 | 屏幕像素 | 当前元素字体大小 | 根元素字体大小 |
| 嵌套影响 | 无 | 有 | 无 |
| 响应性 | 否 | 是 | 是 |
| 计算复杂度 | 简单 | 复杂 | 简单 |

## 实际应用建议

### 设置根元素字体大小
```css
html {
    font-size: 16px;  /* 可选：显式设置根字体大小 */
}

/* 或使用媒体查询实现响应式字体 */
@media (max-width: 768px) {
    html {
        font-size: 14px;
    }
}

@media (min-width: 1200px) {
    html {
        font-size: 18px;
    }
}
```

### 组合使用
```css
/* 用rem设置整体布局 */
.container {
    width: 60rem;      /* 响应式宽度 */
    padding: 2rem;     /* 响应式内边距 */
}

/* 用em设置与字体相关的间距 */
h1 {
    font-size: 2rem;   /* 大标题 */
    margin-bottom: 0.5em;  /* 基于字体大小的间距 */
}

/* 用px设置固定元素 */
.border {
    border: 1px solid #ccc;  /* 固定边框 */
}
```

## 兼容性

- px: 所有浏览器都支持
- em: IE6+支持
- rem: IE9+支持（IE8及以下需要polyfill）

在现代Web开发中，推荐使用rem进行响应式设计，结合px用于固定尺寸元素，这样可以兼顾响应性和精确控制。
