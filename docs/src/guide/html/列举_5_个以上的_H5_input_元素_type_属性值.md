# 列举 5 个以上的 H5 input 元素 type 属性值？（必会）

HTML5 中 input 元素的 type 属性新增了许多类型，以下是常见的 H5 input 元素 type 属性值：

### 1. text
- 普通文本输入框
- `<input type="text">`

### 2. email
- 邮箱地址输入框
- 自动验证邮箱格式
- `<input type="email">`

### 3. url
- URL 地址输入框
- 自动验证 URL 格式
- `<input type="url">`

### 4. tel
- 电话号码输入框
- 移动端会显示数字键盘
- `<input type="tel">`

### 5. number
- 数字输入框
- 移动端会显示数字键盘
- 可设置 min、max、step 属性
- `<input type="number">`

### 6. range
- 滑动条控件
- 用于选择数值范围
- `<input type="range">`

### 7. date
- 日期选择器
- 显示日历控件
- `<input type="date">`

### 8. time
- 时间选择器
- 选择小时和分钟
- `<input type="time">`

### 9. color
- 颜色选择器
- 显示颜色选择面板
- `<input type="color">`

### 10. search
- 搜索框
- 某些浏览器会显示清除按钮
- `<input type="search">`

这些新的 input 类型不仅提供了更好的用户体验，还提供了内置的验证功能和更适合的虚拟键盘（在移动设备上）。
